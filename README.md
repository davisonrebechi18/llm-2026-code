# Projeto

O RAG resolve esses problemas fornecendo ao LLM documentos relevantes no momento da pergunta. Não esperamos que o modelo tenha memorizado a resposta. Recuperamos a informação correta e a entregamos ao LLM, e o modelo gera uma resposta fundamentada. Isso nos permite inserir conhecimento que o modelo nunca viu durante o treinamento. É por isso que o RAG ainda é a forma mais comum de usar LLMs na indústria.

Para tornar isso concreto, criamos um agente de perguntas frequentes (FAQ) para o nosso curso. Um aluno pergunta algo como "quando o curso começa?" e o agente responde com base nos dados das perguntas frequentes que preparamos.
